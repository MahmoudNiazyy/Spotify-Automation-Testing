Kindly is a Java-based automation framework using Selenium for automating tasks on Spotify.com. This README provides a step-by-step guide on how to set up and run the automation for the following scenarios:

Signup on Spotify.com
Login to Spotify.com
Add a Playlist on Spotify.com
Run the Playlist
Prerequisites
Java Development Kit (JDK) 8 or higher
Maven 3.6.0 or higher
Selenium WebDriver
ChromeDriver (or any other driver for your preferred browser)
A text editor or IDE (e.g., Eclipse, IntelliJ IDEA)

Automation Workflow
1. Signup on Spotify.com
The automation will navigate to Spotify.com and fill out the signup form using the provided email and password.

2. Login to Spotify.com
After signing up, the automation will log in using the specified credentials.

3. Add Playlist on Spotify.com
Once logged in, it will create a new playlist with the specified name and add the desired song.

4. Run the Playlist
Finally, the automation will play the created playlist.
